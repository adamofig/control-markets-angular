The Modular Creative Engine

Control Markets is a powerful open-source platform that enables the creation of complex agentic workflows through a visual node-based interface. Think of it as a combination of the N8N flexibility, the ease of Flowise, and the depth of Persona-Based Agents. It provides a unique environment for marketing automation and content generation.

Dual-Layer Architecture

A clean infinite canvas for orchestration paired with a powerful, draggable detail layer for deep configuration and business logic.

Multi-Modal AI

Native integration with Google Cloud AI (Gemini, Veo, Vertex AI), OpenAI, and Anthropic for text, image, and video generation.

Node-Based Orchestration

Connect Input (Assets), Process (Agents, Tasks), and Output (Channels) nodes to build your workflow.

Built with Angular 20, Ionic 8, Nest JS, and Firebase. Bring your own API keys and own your creativity.

Where We Stand

Control Markets is currently in Alpha. It is functional, usable, and ready for you to build your own flows today. While we're still polishing features and refining the architecture, the core engine is stable enough for exploration and early adoption.

There is no promotion for this project yet, until the beta version, if you're lucky enough to discover it, know what to expect. In the next phase, I'll focus on comprehensive documentation, bug fixes, and feature expansion. I invite you to join this open-source journey: try the demo, star the project on GitHub to track progress, or contribute as a collaborator as we work toward our first stable release.
